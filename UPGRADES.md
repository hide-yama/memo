## 更新履歴

### 2026-02-15
- **カテゴリ分け機能**: frontmatterに `category` フィールド追加、カテゴリ別ページ・ナビゲーション作成
- **公開日スケジューリング**: 未来日付の記事を自動非表示、Vercel cronで毎日リビルド
- **サムネイル付きカードレイアウト**: PostCardをサムネイル上・タイトル下のカード型に変更、画像未設定時のmemoプレースホルダー
- **レスポンシブ調整**: スマホ1カラム / デスクトップ2カラム、サムネイル比率3:2
- **タグUI改善**: 横スクロール化、先頭に「すべて」ボタン追加
- **記事詳細ページ改善**: サムネイル表示、本文フォント縮小（18px→15px）、タイトル文字サイズ縮小、話者名後の改行処理
- **プレビューモード**: `/preview` でパスワード認証後に未来記事を含む全記事表示
- **タグページバグ修正**: 日本語タグのURLデコード処理追加
- **読了時間の表示**: 日本語500文字/分で算出、カード・記事詳細に表示
- **シリーズナビ（前後の記事リンク）**: 記事詳細フッターに前後ナビ
- **関連記事の表示**: タグ共通数ベースで2件表示

## 読者体験の強化

- [ ] **目次（Table of Contents）** — 記事詳細にh2/h3から自動生成する目次。長文記事のナビゲーション改善
- [ ] **ページ内スクロール進捗バー** — 画面上部に読了進捗を示すプログレスバー
- [ ] **ページネーション** — 記事数が増えた際の一覧ページ分割（または無限スクロール）
- [ ] **月別アーカイブ** — `/archive/2026-02` のような月別記事一覧ページ
- [ ] **パンくずリスト** — ホーム > カテゴリ > 記事 のナビゲーション
- [ ] **ダークモード** — システム設定連動 + 手動切替トグル

## SEO・ディスカバリ

- [ ] **RSS / Atomフィード** — `/feed.xml` で配信、RSSリーダー対応
- [ ] **サイトマップ自動生成** — `/sitemap.xml` をビルド時に生成
- [ ] **OGP画像の自動生成** — タイトル・カテゴリからOG画像を動的生成（`next/og` or Satori）
- [ ] **構造化データ（JSON-LD）** — 記事ページにArticleスキーマを埋め込み、リッチリザルト対応
- [ ] **canonical URL** — 各ページに正規URLメタタグを設定

## SNS・エンゲージメント

- [ ] **SNSシェアボタン** — X(Twitter)・LINE・はてブなどのシェアリンク
- [ ] **コメント機能** — giscus（GitHub Discussions）等の外部サービス連携
- [ ] **記事のいいね / リアクション** — 匿名で押せるリアクションボタン

## コンテンツ・UI

- [ ] **検索の強化** — 本文全文検索対応（現在はタイトル・説明・タグ・カテゴリのみ）
- [ ] **人気記事ランキング** — PV数ベースまたは手動キュレーションの人気記事セクション
- [ ] **記事内の画像ギャラリー / Lightbox** — 画像クリックで拡大表示
- [ ] **コードブロックのコピーボタン** — 技術記事向け、ワンクリックコピー
- [ ] **外部リンクカード（embed）** — URLを貼ると自動でOGP情報を取得しカード表示
